GAS

  • GAS
    • (def)
      • Google が提供する各種サービスの自動化・連携を行うためのローコード開発ツール。
        • *ローコード開発: 可能な限りソースコードを書かずに開発する手法。
      • ブラウザ上で開発。
      • JS をベースに作られている。一部、GAS 専用の関数が追加されている。
      • API 経由で外部のサービス・アプリなどと連携可能。
      • 注意点：
        • 実行時間に制限がある。（有料 Google Workspace アカウントでも同様）
          • スクリプト実行時間：　６分（１回の実行につき）
          • カスタム関数実行時間：　３０秒（１処理につき）
        • ローカルファイルの操作は行えない。
    • Use cases
      • *GASを使って開発したプログラムを、Web アプリとして公開することも可能。
      • Google スプレッドシートのデータを Gmail に取り組んでメールを
        自動作成・送信。
      • Google Drive 上のファイルが更新されたことを Gmail / チャットで通知する。
      • Google Forms の自動作成、指定アドレスに入力内容を自動送信。
      • Scraping により収集した情報を Google Spreadsheet に登録。
      • 指定時間に Google Sheets に登録した Slack / Chatwork / 又は会計システムに送信する。
    • 2種類の開発方式
      • *2 types of scripts used with Google Apps
      • Container-Bound scripts
        • *コンテナバインド型
          • Googleサービスに紐づくプログラム開発に使われる方式。
          • Googleサービスと連携可能。Webアプリ開発可能。
        • (doc) Scripts created from a Google Apps (Sheets/Docs/Slides/Forms) file. Said scripts are bound (attached) to its container (a Google Apps file).
          • Cannot be detached from the file they are bound to.
          • Created by selecting Extensions > Apps Script within the Google App.
      • Standalone scripts
        • *スタンドアロン型
          • Googleサービスに紐づかないプログラム開発に使われる方式。
          • Googleサービスと連携可能。Webアプリ開発可能。
        • (doc) Scripts not bound to a Google Apps (Sheets/Docs/Slides/Forms) file.
          • Created via Google Drive / script.google.com
    • Shortcuts
      • Cmd + D → Selects the next occurence of the current selection (highlighted bit).

  • Built-in Classes (GAS用 組み込みオブジェクト)
    • (def)
      • *On 「 組み込みオブジェクト」:

      → “JSに組み込まれているクラスを、組み込みオブジェクトと言います” (本当に？) (引用)

    • SpreadsheetApp
      • (def)
        • (doc) Access and create Google Sheets files.
      • .getActiveSpreadsheet()
        • (doc) Returns the currently active spreadsheet, or null if there is none.
          • Return type: Spreadsheet
        • Generally preferred over .openById() .
      • .openById()
        • (doc) Opens the spreadsheet with the given ID. A spreadsheet ID can be extracted from its URL.
          • The spreadsheet ID in the URL https://docs.google.com/spreadsheets/d/abc1234567/edit#gid=0 is "abc1234567".
          • Return type: Spreadsheet
      • .openByURL()
        • (doc) Opens the spreadsheet with the given URL.
          • Return type: Spreadsheet
    • Spreadsheet
      • (def)
        • (doc) Access and modify Google Sheets files. Common operations are adding new sheets and adding collaborators.
      • getSheetByName()
        • (doc) Returns a sheet with the given name.
          • Return type: Sheet
          • Returns null if there is no sheet with the given name.
      • getActiveSheet()
        • (doc) Gets the active sheet withinin a spreadsheet.
          • Return type: Sheet
    • File
      • (def)
        • (doc) A file in Google Drive.
      • .getName()
        • (doc) Gets the name of the File.
        • Return type: String

  • Glossary
    • Spreadsheet
      • (gpt) Refers to the entire document / file that contains one or more sheets (aka tabs/worksheets)
    • Sheet
      • (gpt) Refers to an individual tab / worksheet within a spreadsheet.
    • 組み込みオブジェクト
      • “JSに組み込まれているクラスを、組み込みオブジェクトと言います” (本当に？)
        • 引用
    • Active sheet
      • (doc) Sheet that is being displayed in the spreadsheet UI.
      • (gpt) Sheet that is currently being viewed / displayed by the user.
